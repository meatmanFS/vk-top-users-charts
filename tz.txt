Тестовое задание. 

Задача: модуль визуализации статистических данных.

Требования: 
• Нужно найти апи по выгрузке данных на свой выбор (например VK API и выгрузка пользователей VK). 
• Организовать систему импорта данных с этого апи. 
• Импортировать не менее 3 миллионов записей (например пользователей VK). Построить таблицу соответствия по любым двум критериям (например отношение кол-ва зарегистрированных пользователей ко времени). 
• Построить графики по данным критериям (использовать хотя бы 2 типа графиков).

Стек: 
PHP/Laravel or NodeJS + React.js, любая база данных на выбор (MySQL, MongoDB, etc.)

Фронтент:
1. Пользователь 
a. Видит таблицу с данными 
b. Видит график над таблицей 
c. На график можно вывести данные из любой колонки таблицы
d. Сортировать таблицу по любому из полей
e. Менять формат группировки данных (вывод данных сгруппированных по неделям/ дням/месяцам) (опционально)
Бекенд:
1. Система импорта данных из любого источника с их дальнейшим сохранением в базу (API/парсин)
